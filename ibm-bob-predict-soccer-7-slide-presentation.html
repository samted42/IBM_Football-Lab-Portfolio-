  • Tasks 1–2: Install libraries · Download 49,000+ match dataset (1872–2026)
  • Task 3: Explore tournaments, top teams, decade-by-decade match distribution
  • Task 4: Engineer 8 features — win rate, goals, recent form (last 10), venue, tournament. Zero data leakage
  • Task 5: Time-based split — train on matches before 2018, test on 2018–2026
  • Task 6: Train Random Forest (200 trees, depth 12). Result: 52% accuracy vs 45% baseline
  • Tasks 7–8: Save model · Test predict_match() — Brazil vs Argentina, Germany vs Brazil
  • Tasks 9–10: Write Streamlit web app · Launch at http://localhost:8501
